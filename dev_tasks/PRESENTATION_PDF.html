Часто задаваемый вопрос
Демо-данные vs реальные
📋 Сейчас (демо-режим)
• Цифры охватов и лидов — примерные
• Подписчики на платформах — mock
• Расписание постов — демонстрационное
• Видео в очереди — симуляция
Нужно для презентации — показывает как будет выглядеть платформа в работе
🚀 После полного запуска
• Реальные просмотры с TikTok / Instagram API
• Реальные лиды из форм и соцсетей
• История реальных публикаций
• Живая аналитика по дням / неделям / месяцам
Архитектура готова — нужно только подключить боевые API

Так работает любой SaaS-продукт: сначала строится интерфейс и логика, затем подключаются реальные данные. Figma, Notion, Trello — все начинали именно так. Платформа полностью готова к подключению боевых данных.
